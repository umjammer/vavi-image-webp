# javavp8decoder

An implementation of the VP8 image/video codec in _**pure Java**_.

## Install

https://jitpack.io/#umjammer/javavp8decoder

## Usage

```java
    BufferedImage image = ImageIO.read(Paths.get("/foo/bar.webp").toFile());
```
